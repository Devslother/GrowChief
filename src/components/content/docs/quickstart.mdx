<H2>Requirements</H2>
<Divider />
<P>Before installing GrowChief, ensure you have the following:</P>
<List>
  <Item>
    Docker and Docker Compose ([Installation
    guide](https://docs.docker.com/engine/install/))
  </Item>
  <Item>Git</Item>
  <Item>The following ports available: 5002, 8080</Item>
</List>

<H2>Setup</H2>
<Divider />
<H3>1. Clone the Repository</H3>
<P>Run the following command to clone the repository:</P>
<CopyableCode>{`git clone https://github.com/growchief/growchief.git`}</CopyableCode>

<H3>2. Configure the Auth Secret</H3>
<P>
  Open docker-compose.yml and set the AUTH_SECRET value to a secure, random
  string.
</P>

<H2>First Start-Up</H2>
<Divider />

<P>To launch GrowChief, run:</P>
<CopyableCode>{`docker compose up -d`}</CopyableCode>
<P>Wait a few moments for all services to start.</P>
<P>
  Once running, open your browser and go to:
  [http://localhost:5002/](http://localhost:5002/)
</P>
