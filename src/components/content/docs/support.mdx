<P>Sometimes, things can go wrong, or you need some help!</P>
<P>
  Note that the Self Hosted version of GrowChief is supported by the community
  in their free time, on a best-efforts basis. Please post your question and be
  patient.
</P>

<List>
  <Item>
    <b>Discord – </b> Flexible chat, with screenshots and screen sharing,
    probably the best option for support.
  </Item>
  <Item>
    <b>GitHub issue – </b> backup option if you are unable to use Discord.
  </Item>
</List>

<H2>How to effectively ask for support</H2>

<Divider />

<P>
  Try to follow this guide when asking for support, in this order; Goal,
  Environment, Changes, Results.
</P>

<List>
  <Item>
    <b>Goal: </b> Start off by explaining what you were trying to do
    <List>
      <Item>I want to schedule a post on Reddit at 6pm</Item>
      <Item>I want to run in a Linux container on a Raspberry Pi</Item>
      <Item>I want to use a custom domain name</Item>
    </List>
  </Item>
  <Item>
    <b>Environment: </b> Share the relevant parts about your environment. Web
    App issues; Are you using Firefox, Chrome, etc? Installation/Other issues; a
    Mac, Linux, Windows, how did you install?
    <List>
      <Item>I’m using Firefox on Windows 10</Item>
      <Item>
        I’m using a Raspberry Pi 4 with Ubuntu 20.04, and Node version 18
      </Item>
      <Item>This is a new installation on a Mac</Item>
    </List>
  </Item>
  <Item>
    <b>Changed: </b> Most likely something has changed, what is it?
    <List>
      <Item>I updated my browser to the latest version and now …</Item>
      <Item>I found a change in the latest version and now …</Item>
      <Item>I think this used to work, but now…</Item>
    </List>
  </Item>
  <Item>
    <b>Results: </b> What happened? What did you expect to happen?
    <List>
      <Item>I see a blank screen</Item>
      <Item>I see an error message</Item>
      <Item>I see a 404 page</Item>
    </List>
  </Item>
</List>
